jose-java
=========

Json Web Token (JWT) Library for Java

### Specification

* http://self-issued.info/docs/draft-ietf-oauth-json-web-token.html

### Required library  

* glassfish JSON Processing RI jar (javax.json-1.0.4.jar)
  * https://jsonp.java.net/download.html

### Json Web Token
#### Supported Algorithm

* HMAC-SHA256
* HMAC-SHA384
* HMAC-SHA512
